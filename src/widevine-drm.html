--- layout: solution title: Widevine DRM permalink: /solutions/widevine-drm summary_html: >-

Widevine’s DRM solution provides the capability to licese, securely distribute and protect playback of content on any consumer device. Content owners, multiple service operators and digital media providers can utilize Widevine’s solutions to ensure revenue generating services keep flowing to whatever device consumers desire.

features: - title: Content Protection icon_path: '/assets/img/icon-content-protection.svg' summary_html: >-

Provides ubiquitous device coverage and extends synergy across multiple content protection systems. See a list of currently supported platforms.

- title: Video Playback icon_path: '/assets/img/icon-video-playback.svg' summary_html: >-

Fully featured HTML5 video player with adaptive streaming, QoS, and accessibility support across devices.

- title: Standardized Format icon_path: '/assets/img/icon-standardized.svg' summary_html: >-

Encrypt once, play anywhere utilizing industry adopted media containers including ISO BMFF 9i.e. MP4) and WebM.

- title: Legacy System icon_path: '/assets/img/icon-legacy-system.svg' summary_html: >-

Support for concurrent access to all content protection systems. Providing migration tools and flexibility while adopting new technologies

- title: Device Security icon_path: '/assets/img/icon-device-security.svg' summary_html: >-

Robust device security is enabled by using factory provisioned keyboxes to establish a hardware root of trust, secure decryption and content rendering.
