Welcome to the CWIP Portal

Existing CWIP Partners can use the following can access commonly used resources via the links below:

Documentation

All of Widevine's updated and most recent documenation can be found on the official Widevine documenation site. Available documentation is contingent upon licensing Widevine and the access level granted.

Go

Widevine Discussion Group

Widevine's official Discussion Group for the CWIP Certified community. This discussion group is a place where partners can discuss Widevine related topics, ask questions, and seek assistance from other community memebers and Widevine moderators.

Go

Create A Support Case

For help with technical issues or questions, CWIP Partners can submit a case. Please provide as much information as possible including:

  • A detailed description of the issue.
  • Specific error messages.
  • Logs/log messages.
  • Detailed platform information (OS + version, build number, etc).
  • Exact steps to reproduce the issue.

Go

Request Service Credentials

Complete the form to request access for a single organization (portal) to the Widevine License Service on Google.
